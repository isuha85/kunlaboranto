### 결론

단락나누기는 "Enter 2번" 인데, 원하는 방식이 아니다.<br>  
줄만바꾸기위해서는  
문장뒤의 "공백2개" 가 표준이나. *.md 파일에서 일괄적으로 뒤의 공백을 없애는 경우, 표멧이 망가진다. &lt;br&gt; 을 추가하는 것이 가장 안전하다.<br>
또는 "공백2개"로 작성하고 최종배포단계에서 정규식으로 모두 &lt;br&gt; 로 변경한다.

-----

### From Gemini

마크다운(Markdown)에서 줄바꿈은 일반적인 문서 도구와 약간 다르게 작동합니다. 단순히 `Enter`를 한 번 치는 것만으로는 줄바꿈이 되지 않는 경우가 많기 때문인데요, 상황별로 3가지 방법을 정리해 드립니다.


### 1\. 가장 많이 쓰이는 방법: 스페이스 2번 + Enter

문장 맨 끝에 **공백(Space)을 두 번** 넣고 `Enter`를 누르는 방식입니다. 마크다운의 공식적인 줄바꿈(Line Break) 규칙입니다.

  * **방법:** `문장 끝(스페이스)(스페이스)(Enter)`
  * **특징:** 줄 사이의 간격이 좁게 유지됩니다. (같은 단락 안에서의 줄바꿈)

-----

### 2\. 단락 나누기: Enter 2번

가장 직관적인 방법으로, 줄 사이에 **빈 줄**을 하나 만드는 방식입니다.

  * **방법:** `문장 끝(Enter)(Enter)`
  * **특징:** 줄 사이의 간격이 넓어지며, 새로운 \*\*단락(Paragraph)\*\*이 시작됩니다.

-----

### 3\. HTML 태그 사용: `<br>`

어떤 상황에서도 강제로 줄을 바꾸고 싶을 때 사용합니다.

  * **방법:** `문장 끝<br>다음 문장`
  * **특징:** 공백 입력이 헷갈리거나, 여러 줄을 연속으로 띄우고 싶을 때 편리합니다.

-----

### 💡 한눈에 비교하기

| 방법 | 입력 예시 | 결과 형태 |
| :--- | :--- | :--- |
| **스페이스 2번** | `안녕하세요..` (점 대신 공백) | 줄 간격이 좁음 (같은 단락) |
| **Enter 2번** | `안녕하세요`<br><br>`반갑습니다` | 줄 간격이 넓음 (새 단락) |
| **`<br>` 태그** | `안녕하세요<br>반갑습니다` | 강제 줄바꿈 |

-----

### **VS Code 팁**

만약 매번 스페이스를 두 번 치는 게 귀찮다면, VS Code 설정에서 \*\*"Render Whitespace"\*\*를 켜보세요. 공백이 점(`.`) 형태로 보여서 줄바꿈이 제대로 들어갔는지 확인하기 훨씬 편해집니다\!

스니펫 만드실 때도 `body` 안에 `\n`이나 빈 문자열 `""`을 넣어서 자동으로 줄바꿈이 포함되게 구성하실 수도 있어요. 유용하게 활용해 보세요\!
